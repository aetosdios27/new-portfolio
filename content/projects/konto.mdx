// what
Konto is a compact ledger-oriented project for modeling financial state without pretending money is a scalar.

// why
Financial systems fail when mutation, identity, and auditability are treated as afterthoughts.

// how
The implementation keeps account state explicit, tracks transitions as first-class records, and treats reconciliation as core behavior.
