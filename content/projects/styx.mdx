// what
Styx is a desktop-first system for turning local project context into a usable operating surface.

// why
Most developer tools either hide too much state or expose it without shape. Styx exists to keep the state close to the work.

// how
The hard parts live in process isolation, async orchestration, and rendering a responsive UI while background tasks continue to stream state.
