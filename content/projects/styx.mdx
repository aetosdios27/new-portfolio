A Tier 3 production-grade BitTorrent client built v2-native (BEP 52) from day one. Engineered as a multi-crate Rust monorepo for uncompromising privacy, Styx ships with SHA-256 Merkle trees, ephemeral peer IDs, and an embedded ML pipeline (LightGBM + PyTorch via ONNX Runtime) for adaptive peer throttling prediction without runtime Python dependencies. The entire engine is compiled into a single Rust binary, featuring both a Tauri GUI and a ratatui TUI for extreme operational flexibility.
