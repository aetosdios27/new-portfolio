// what
Iris is an interface experiment around inspecting visual state, metadata, and small transformations in one place.

// why
Visual tooling gets slow when every operation requires switching context.

// how
The scaffold favors direct manipulation, local previews, and a narrow processing pipeline that can be swapped without rewriting the UI.
