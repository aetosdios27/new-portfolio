Zero-Copy GPU Image Viewer

// bullet

Built a zero-CPU-copy image display pipeline on Linux using raw Vulkan external memory (ash) and DMA-BUF, eliminating all host-side pixel transfers in the hot path.

// bullet

Reconstructed the rendering backend from wgpu to raw Vulkan after identifying ownership model incompatibilities with external memory handles.
