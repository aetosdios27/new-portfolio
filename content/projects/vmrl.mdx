A decentralized cryptographic ledger designed to anchor software release metadata directly onto the Base Sepolia network. VMRL utilizes a custom Solidity smart contract and a viem-powered TypeScript/Bun CLI to bind Git commit hashes to signed on-chain receipts, creating a trustless, immutable audit trail that mathematically guarantees the provenance and integrity of open-source packages.
