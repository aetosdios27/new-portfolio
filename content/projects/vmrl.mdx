On-Chain Git Release Provenance

// bullet

Designed and deployed a Solidity smart contract to Base Sepolia that anchors git release hashes on-chain, enabling trustless provenance verification for open-source packages.

// bullet

Built a TypeScript/Bun CLI and Next.js web explorer using viem for contract interaction and release history lookup.
