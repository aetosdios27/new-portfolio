# new-portfolio

Brutalist terminal portfolio scaffold for `aetos`, built from `portfolio-prd.md`.

## Run

```bash
bun install
bun run dev
```

## Structure

- `src/app` contains the App Router routes.
- `src/components` contains the persistent two-panel shell, hologram, patterns, and EPV list view.
- `content/projects` and `content/blog` contain MDX-like source files with frontmatter.
- `public/ascii/pfp.txt` is the placeholder build-time ASCII artifact.
